How It Works

1

Chain-of-Thought Prompting

A Qwen-7B language model reasons step-by-step with few-shot examples covering easy, medium, and hard puzzles.

2

Deterministic Verifier

An AST-based evaluator checks every expression — no eval(), no injection. The reward signal is mathematically exact.

3

Monte Carlo Tree Search

When the LLM fails, MCTS explores the expression tree using UCB selection. Random rollout alone reaches 58% solve rate — no GPU needed.

4

GRPO Fine-Tuning

Successful trajectories train the model via Group Relative Policy Optimization — no critic network, just verified solutions.
